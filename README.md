## 🚀 طريقة التشغيل

### ⚡ بدون سيرفر (مباشرة)
افتح `index.html` في المتصفح — يعمل 100% بدون إنترنت أو سيرفر!

### 🖥️ مع XAMPP/WAMP
1. انسخ المجلد كاملاً إلى `htdocs/pos/`
2. افتح phpMyAdmin وشغّل `database.sql`
3. عدّل `db.php` ببيانات قاعدتك
4. افتح `http://localhost/pos/`

## ✨ المميزات الجديدة (v4.0)

### 🖼️ رفع الصور
- **شعار المطعم**: الإدارة → اضغط على منطقة الشعار → اختر JPG/PNG
- **صور المنتجات**: إضافة منتج → اضغط على منطقة الصورة → اختر من Gallery
- الشعار يظهر **دائرياً** في البون والفاتورة الشهرية

### 💳 الديون التلقائية
- عند اختيار **"دين"** كطريقة دفع → يُسجَّل الدين تلقائياً باسم العميل
- يجب اختيار عميل قبل الدفع بالدين

### 🖨️ الطباعة
- البون: يفتح نافذة منفصلة 80mm — صفحة واحدة فقط
- الشعار الدائري يظهر أعلى كل بون وفاتورة

### 💰 الصندوق
- يزيد تلقائياً عند كل بيع
- ينقص عند تسجيل مصروف
- يزيد عند إيداع أو تسديد دين
- عرض رأس المال + الرصيد الحالي

### 📉 الأيام/الأشهر الضعيفة
- التقارير → يظهر جدول **أضعف 5 أيام** مبيعاً
- مخطط بياني للأشهر مع تمييز الشهور الضعيفة

### 👷 صلاحيات الموظف
- الموظف يرى فقط: **نقطة البيع + الطلبات**
- المدير يرى كل شيء
